# 編集禁止ファイル

以下のファイルは直接編集しないこと。

## 自動生成ファイル

- `build/`、`dist/`、`out/` など、ビルドツールが出力するディレクトリ配下のファイル
- コード生成ツールやスキーマから自動生成されたファイル
- パッケージマネージャーのロックファイル（`package-lock.json`、`yarn.lock`、`Gemfile.lock` など）

## 環境設定・機微情報ファイル

- `.env`、`.env.*` など環境変数を含むファイル
- `local.properties` などローカル固有の設定ファイル
- 秘密鍵、証明書、トークンを含むファイル
