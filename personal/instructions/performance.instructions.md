# パフォーマンス・コンテキスト管理

## MCP サーバー

- 同時有効 MCP サーバー: 10 以下
- 有効ツール総数: 80 以下
- 使わない MCP は無効化する
- プロジェクトごとに必要な MCP だけ有効にする

## コンテキストウィンドウ

- 長大なファイルを丸ごと載せない（必要箇所のみ読む）
- 大きなデータはサブエージェントに処理させる
- セッションが長くなったら要約して新セッションに引き継ぐ

## モデル選択

- 設計判断・レビュー → 高性能モデル
- 定型操作・検索・サブエージェント → 高速モデル
