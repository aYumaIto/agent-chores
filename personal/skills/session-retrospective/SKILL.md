# セッション振り返り

## 利用する場面

- セッション完了後にスキル・ルールの改善ポイントを洗い出したいとき

## 手順

1. 対象期間のセッション会話履歴を取得する
2. 以下のシグナルを探す：
   - ユーザーからの修正指示
   - 同じ指示の繰り返し
   - 3回以上のループ
   - ユーザーによる手動補正
3. シグナルごとに「問題・原因・改善案・対象ファイル」をまとめて提示する（最大3件）
4. ユーザー承認後に対象ファイルを編集する

## 条件

- 一度きりの特殊ケースは対象外
- Finding がなければ「改善シグナルなし」と報告して終了
